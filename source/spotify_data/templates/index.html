Welcome to Spotify Data Visualization page!

Web application is used to data visualisation based on all the “Top 200” and “Viral 50” charts published globally by Spotify since 01.01.2017 to 31.12.2021. Data are stored in PostgreSQL relational database. App has a feature of presenting user-selected data on charts and choropleth maps and creating an account. The application is still under development, more features will be added soon. You can analyze the data yourself using application functionalities.
